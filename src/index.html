About

WanMask is a bridge that allows you to visit the distributed web of tomorrow in your browser today. It allows you to run Wanchain dApps right in your browser without running a full Wanchain node.

WanMask includes a secure identity vault, providing a user interface to manage your identities on different sites and sign blockchain transactions.

You can install the WanMask add-on in Chrome .

Our mission is to make Wanchain as easy to use for as many people as possible.
